# Salva la Pelota - Juego en PyGame
Proyecto desarrollado en Python usando la librería PyGame.

# Descripción
El jugador controla una plataforma que debe evitar que la pelota caiga.
La pelota rebota por el entorno y destruye bloques para ganar puntos.

# Controles
- Debes usar las flechas del teclado para mover plataforma -> <-
- ENTER iniciar juego
- R reiniciar

# Características
- Sistema de menú
- Colisiones con bloques
- Sistema de puntaje
- Estados del juego (inicio, jugando, game over)
- Funciones personalizadas

# Tecnologías
- Python
- PyGame
